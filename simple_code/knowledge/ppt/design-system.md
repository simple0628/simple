# PPT 设计系统

> AI 生成 PPT 时的核心设计规范。遵循这些规则，确保输出专业、美观、一致。

## 1. 视觉氛围

在开始设计之前，先确定整体氛围：

| 氛围 | 关键词 | 适用场景 |
|------|--------|----------|
| 专业严谨 | 克制、数据驱动、留白 | 咨询报告、投资分析、高管汇报 |
| 科技前沿 | 暗色、渐变、发光、网格 | AI产品、技术分享、开发者大会 |
| 学术清晰 | 白底、结构化、引用规范 | 论文答辩、科研报告、课题申报 |
| 政务大气 | 蓝色系、稳重、对称 | 政府汇报、规划方案、招商引资 |
| 活力现代 | 大胆配色、几何图形、动感 | 产品发布、公司介绍、营销提案 |
| 温暖治愈 | 柔和色调、圆角、手写体 | 教育培训、心理学、公益 |

## 2. 配色规则

### 60-30-10 法则
- **60% 主色**：背景或大面积使用（通常是白色或深色）
- **30% 辅色**：标题栏、卡片、分区背景
- **10% 强调色**：按钮、图标、关键数据、装饰线

### 对比度
- 标题文字与背景对比度 ≥ 4.5:1（WCAG AA）
- 正文文字与背景对比度 ≥ 7:1
- 深色背景用白色/浅色文字，浅色背景用深色文字

### 配色方案参考
| 风格 | 主色 | 辅色 | 强调色 | 适用 |
|------|------|------|--------|------|
| 商务蓝 | #FFFFFF | #005587 | #F5A623 | 咨询、汇报 |
| 科技暗 | #0D1117 | #1E40AF | #D4AF37 | AI、科技 |
| 学术白 | #FFFFFF | #003366 | #CC0000 | 答辩、研究 |
| 政务蓝 | #FFFFFF | #0050B3 | #00B4D8 | 政府、规划 |
| 活力红 | #F5F5F7 | #DE3545 | #F0964D | 产品、营销 |
| 自然绿 | #FFFFFF | #2D6A4F | #D4A373 | 环保、健康 |
| 四色科技 | #FFFFFF | #4285F4 | #FBBC04 | 科技、数据 |
| 暗色咨询 | #0D1117 | #7C3AED | #D4AF37 | 董事会、战略 |

### 透明度语义
| 透明度 | 用途 |
|--------|------|
| 3-5% | 背景网格线、极淡装饰 |
| 8-12% | 几何装饰图形、水印 |
| 15-25% | 半透明卡片、遮罩 |
| 40-50% | 辅助文字、次要元素 |
| 100% | 主要文字、关键元素 |

## 3. 字体规则

### 字体选择
- 中文优先：Microsoft YaHei（微软雅黑）
- 英文优先：Arial / Segoe UI / Helvetica Neue
- 最多使用 2 种字体族（标题 + 正文）
- 正文不用粗体

### 字号层级
| 层级 | 字号 | 用途 |
|------|------|------|
| 超大标题 | 44-56px | 封面主标题 |
| 页面标题 | 28-36px | 章节页标题 |
| 区域标题 | 22-24px | 内容页标题 |
| 小标题 | 18-20px | 卡片标题、副标题 |
| 正文 | 14-16px | 段落、要点 |
| 辅助文字 | 11-12px | 页码、注释、来源 |
| 数据高亮 | 36-44px | KPI 数字、关键指标 |

### 行间距
- 标题：1.2-1.3 倍
- 正文：1.5-1.6 倍
- 要点列表：1.8 倍（含段后间距）

## 4. 间距系统

### 页面边距
- 标准边距：60px（所有方向）
- 安全区域：x: 60-1220, y: 60-660（1280×720 画布）

### 元素间距
| 间距类型 | 数值 | 说明 |
|----------|------|------|
| 标题与内容 | 20-30px | 标题下方到内容区 |
| 要点间距 | 12-16px | 列表项之间 |
| 卡片间距 | 20-24px | 并列卡片之间 |
| 卡片内边距 | 20-24px | 卡片内部填充 |
| 图标与文字 | 12-15px | 图标右侧到文字 |
| 页脚上边距 | 40px | 内容到页脚 |

### 间距风格
| 风格 | 圆角 | 内边距 | 间距 | 适用 |
|------|------|--------|------|------|
| 紧凑锐利 | 0-2px | 16px | 12px | 数据密集、咨询 |
| 柔和均衡 | 4-8px | 20px | 16px | 通用商务 |
| 圆润宽松 | 12-16px | 24px | 20px | 科技、现代 |
| 胶囊通透 | 20-30px | 28px | 24px | 创意、轻松 |

## 5. 布局原则

### 页面类型
| 类型 | 用途 | 标题位置 | 内容方式 |
|------|------|----------|----------|
| 封面 | 开场 | 居中或左对齐 | 标题 + 副标题 + 信息 |
| 目录 | 导航 | 顶部 | 编号列表 |
| 章节 | 分隔 | 居中 | 大标题 + 简述 |
| 内容 | 主体 | 顶部标题栏 | 要点/卡片/图表 |
| 结尾 | 收尾 | 居中 | 感谢 + 联系方式 |

### 对齐规则
- 所有元素左边缘对齐（网格系统）
- 标题和内容使用相同左边距
- 居中对齐仅用于封面和章节页
- 数据和指标右对齐

### 每页元素限制
- 最多 5-7 个视觉元素
- 最多 6 个要点
- 每个要点最多 2 行文字
- 留白 ≥ 页面面积的 30%

## 6. 装饰元素指南

### 装饰类型
| 类型 | 何时使用 | 注意事项 |
|------|----------|----------|
| 渐变色块 | 标题栏、背景 | 同色系渐变，角度一致 |
| 几何轮廓 | 封面右侧装饰 | 低透明度(3-15%)，不喧宾夺主 |
| 强调竖条 | 页面左侧 | 4-12px 宽，主色或强调色 |
| 分割线 | 内容分区 | 1-2px，浅灰色 |
| 圆点分隔 | 标题下方 | 小圆(6-12px) + 两侧短线 |
| 三角形 | 角落装饰 | 低透明度，大尺寸 |
| 网格线 | 暗色背景装饰 | 2-3% 透明度 |
| 霓虹线 | 赛博/游戏风 | 双线(粗+细)，发光色 |

### 装饰密度
- 封面：中等装饰（突出标题，营造氛围）
- 章节：少量装饰（聚焦标题）
- 内容：最少装饰（让内容说话）
- 结尾：中等装饰（呼应封面）

## 7. 动画指南

### 转场效果
- 封面 → 第二页：淡入（fade），庄重
- 章节切换：推入（push），有节奏感
- 内容翻页：擦除（wipe），流畅
- 全程统一，不混搭超过 2 种转场

### 入场动画
- 标题：淡入（fade）
- 要点列表：飞入（fly）或淡入（fade）
- 图表/数据：缩放（zoom）
- 不要给装饰元素加动画
- 一页内动画不超过 3 个元素

## 8. 必须做 / 绝对不做

### 必须做
- 每页有明确的视觉重心
- 配色全程一致（不超过 4 种颜色）
- 标题传达核心信息（不是"第一部分"而是"营收增长 32%"）
- 数据用图表或大号数字突出
- 每张幻灯片能独立表达一个观点

### 绝对不做
- 不要在一页内用超过 3 种字号
- 不要纯白背景 + 浅灰文字（对比度不足）
- 不要给每个元素都加不同的动画
- 不要在内容页堆满装饰（留白 ≥ 30%）
- 不要用艺术字、WordArt、花哨字体
- 不要在要点里写完整段落（每条 ≤ 2 行）
