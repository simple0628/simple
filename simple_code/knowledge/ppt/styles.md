# PPT 风格推荐矩阵

> 根据用户的行业、目标和偏好，推荐最合适的设计风格。

## 快速推荐

| 用户说 | 推荐风格 |
|--------|----------|
| "做个工作汇报" | 商务蓝 或 柔和均衡 |
| "科技风" / "AI风" | 暗色科技 或 极简科技 |
| "学术答辩" / "论文" | 学术白 |
| "给领导看" / "政府汇报" | 政务蓝 或 政务红 |
| "产品发布" / "公司介绍" | 活力现代 |
| "数据分析" / "投资" | 咨询风 或 暗色咨询 |
| "教育" / "培训" | 温暖柔和 |
| "游戏" / "技术分享" | 赛博像素 |
| 没有特别要求 | 商务蓝（最安全） |

## 风格详细定义

---

### 商务蓝（Business Blue）

**氛围：** 专业、可信、克制。适合大多数商务场景。
**最佳场景：** 工作汇报、季度总结、方案提案

**配色：**
| 角色 | 颜色 | 用途 |
|------|------|------|
| 主背景 | #FFFFFF | 页面底色 |
| 品牌蓝 | #005587 | 标题栏、强调条、编号方块 |
| 渐变蓝 | #0076A8 | 渐变终点、辅助装饰 |
| 金橙 | #F5A623 | 强调线、关键数据、装饰点 |
| 标题文字 | #2C3E50 | 标题 |
| 正文文字 | #5D6D7E | 段落、要点 |
| 辅助灰 | #7F8C8D | 页码、注释 |

**字体：** Microsoft YaHei / Arial
**封面：** 白底，左对齐标题，左侧蓝色竖条装饰，右侧几何轮廓
**内容页：** 渐变标题栏 + 金色洞察栏 + 左侧强调条
**间距风格：** 柔和均衡（8px 圆角，20px 内边距）

**避免：** 装饰过多、标题居中（这不是活动海报）

---

### 暗色科技（Dark Tech）

**氛围：** 前沿、深邃、专业感。暗色背景让数据和图形更突出。
**最佳场景：** AI 产品发布、技术分享、开发者大会

**配色：**
| 角色 | 颜色 | 用途 |
|------|------|------|
| 主背景 | #1A1A2E | 深色底 |
| 品牌橙 | #D97757 | 强调线、标签、装饰 |
| 科技蓝 | #4A90D9 | 辅助色、链接、图标 |
| 薄荷绿 | #10B981 | 正面数据、成功状态 |
| 白色文字 | #FFFFFF | 标题 |
| 浅灰文字 | #94A3B8 | 正文 |

**字体：** Arial / Segoe UI
**封面：** 深色渐变底 + 网格线(3%透明度) + 神经网络/发光装饰
**内容页：** 暗色顶条 + 三列卡片(彩色顶边框) + 结论先行
**间距风格：** 圆润宽松（12px 圆角，24px 内边距）

**避免：** 纯白背景、过亮的颜色、没有呼吸感的密集排版

---

### 学术白（Academic Clean）

**氛围：** 严谨、清晰、结构化。用最少的装饰传递最多的信息。
**最佳场景：** 论文答辩、科研报告、课题申报

**配色：**
| 角色 | 颜色 | 用途 |
|------|------|------|
| 主背景 | #FFFFFF | 纯白 |
| 深蓝 | #003366 | 标题栏、编号 |
| 学术蓝 | #0066CC | 辅助、图表 |
| 红色 | #CC0000 | 左侧强调条、关键标注 |
| 标题文字 | #333333 | 标题 |
| 正文文字 | #666666 | 正文 |

**字体：** Microsoft YaHei / Arial（不用花哨字体）
**封面：** 白底居中，深蓝顶条 + 红色左竖条，导师/答辩人信息
**内容页：** 深蓝标题栏(70px) + 浅蓝关键信息条 + 清晰的层级
**间距风格：** 紧凑锐利（2px 圆角，16px 内边距）

**避免：** 过多装饰、圆角卡片、渐变效果、任何花哨元素

---

### 政务蓝（Government Blue）

**氛围：** 大气、稳重、科技感。传达权威性和前瞻性。
**最佳场景：** 政府汇报、规划方案、招商引资

**配色：**
| 角色 | 颜色 | 用途 |
|------|------|------|
| 主背景 | #FFFFFF / 深蓝渐变 | 内容页白底 / 封面深色 |
| 政务蓝 | #0050B3 | 标题栏、编号、强调 |
| 亮蓝 | #00B4D8 | 渐变终点、科技装饰 |
| 深蓝 | #003366 | 封面/章节背景 |
| 文字 | #1A1A1A | 标题 |

**封面：** 深蓝渐变底 + 科技网格 + 几何圆环
**内容页：** 蓝色渐变顶条 + 编号方块 + 底部装饰线
**间距风格：** 柔和均衡

**避免：** 花哨动画、不对称布局、过于活泼的颜色

---

### 活力现代（Vibrant Modern）

**氛围：** 大胆、有活力、视觉冲击力强。
**最佳场景：** 产品发布、公司介绍、营销提案

**配色：**
| 角色 | 颜色 | 用途 |
|------|------|------|
| 主背景 | #F5F5F7 | 浅灰底 |
| 红色 | #DE3545 | 主色、三角装饰、强调 |
| 橙色 | #F0964D | 辅助色 |
| 深灰 | #333333 | 标题、暗色装饰 |

**封面：** 浅灰底 + 大三角形几何切割（左上/左下/右下）
**内容页：** 白色导航栏 + 标题双色装饰 + 白色圆角卡片
**间距风格：** 圆润宽松

**避免：** 过多文字、保守配色、没有几何元素

---

### 赛博像素（Cyber Pixel）

**氛围：** 复古、霓虹、赛博朋克。适合年轻受众和技术社区。
**最佳场景：** 技术分享、编程教学、游戏介绍

**配色：**
| 角色 | 颜色 | 用途 |
|------|------|------|
| 主背景 | #0D1117 | 深空黑 |
| 霓虹绿 | #39FF14 | 主强调色、标题、线条 |
| 赛博粉 | #FF2E97 | 辅助强调 |
| 电光蓝 | #00D4FF | 链接、图标 |
| 金黄 | #FFD700 | 高亮、徽章 |

**封面：** 深黑底 + 霓虹双线(粗+细) + 像素角落方块 + PRESS START
**内容页：** 霓虹双线顶部 + 像素方块角落装饰 + 开放内容区
**间距风格：** 紧凑锐利（0px 圆角）

**避免：** 白色背景、圆角、传统商务排版

---

### 暗色咨询（Dark Consulting）

**氛围：** 高端、神秘、数据驱动。金色点缀传递尊贵感。
**最佳场景：** 董事会汇报、战略规划、投资分析

**配色：**
| 角色 | 颜色 | 用途 |
|------|------|------|
| 主背景 | #0D1117 | 暗色底 |
| 蓝紫渐变 | #1E40AF → #7C3AED | 顶部装饰条 |
| 金色 | #D4AF37 | 竖线、分割线、标签 |
| 白色文字 | #FFFFFF | 标题 |
| 浅灰文字 | #9CA3AF | 正文 |

**封面：** 暗色底 + 蓝紫渐变条 + 左侧金线 + 网格装饰
**内容页：** 白底 + 渐变细条 + Takeaway Bar(金色左边框)
**间距风格：** 柔和均衡

**避免：** 亮色背景、过多颜色、缺少金色点缀

---

### 温暖柔和（Warm Healing）

**氛围：** 治愈、专业而不冰冷、有人文关怀。
**最佳场景：** 心理学讲座、教育培训、咨询案例

**配色：**
| 角色 | 颜色 | 用途 |
|------|------|------|
| 渐变背景 | #1E3A5F → #2E5C8E → #3D8B7A | 封面/章节 |
| 安全蓝 | #2E5C8E | 主色、左侧强调条 |
| 治愈绿 | #3D8B7A | 辅助色 |
| 暖橙 | #E07843 | 装饰线、强调 |
| 正文 | #374151 | 正文 |
| 白色 | #F8FAFC | 内容页背景 |

**封面：** 蓝绿渐变底 + 暖橙细线 + 引用卡片 + 关键词胶囊标签
**内容页：** 左侧强调条(8px) + 圆角卡片(彩色顶条/左边框) + 柔和分割线
**间距风格：** 圆润宽松（16px 圆角）

**避免：** 锐利边角、红色警告色、冰冷的纯蓝/纯灰
